Want

  • First class functions
  • References (parameters, fields, variables etc.) that are not reassignable by default
  • Statically checked types
  • Type lambdas
  • Higher kinded types
  • Private (transitive) dependencies by default
  • Module private members by default
  • Defaulted parameters
  • One[T] <: ?[T]
  • One[T] <: NonEmptySeq[T]
  • NonEmptySeq[T] <: Seq[T]
  • One[T] <: NonEmptySet[T]
  • NonEmptySet[T] <: Set[T]
  • None <: ?[T]
  • None <: List[T]
  • ?[T] <: Seq[T]
  • ?[T] <: Set[T]
  • Seq[T].head: ?[T]
  • NonEmptySeq[T].head: T
  • Convenient ? dereferencing as syntax sugar for flatMap:
    list: List[T] = List[T]()
    x1: One[List[T]] = list.one
    x: ?[List[T]] = x1
    y1:?[T] = x?.head
    y2:?[T] = x.flatMap(_.head)

Do Not Want

  • Null
  • Implicits of any kind
  • Variable infix precedence - either strict left to right, or compiler requires the brackets
  • Primitives that lack a common supertype
